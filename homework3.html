The reason can be a slow connection, an error with the image file, or if the user uses a screen reader.</p> <p> There is a significant lack of image text alternatives or "alt text" throughout the links on the library website. Without alt text, a user with a screen reader would not be able to get information on images, flyers to events or navigate through imaged based links. Example the images on the Home page directing patrons to the yearbook collection does not have alt text for what the image is showing.
